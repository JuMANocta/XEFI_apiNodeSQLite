# API REST NODE JS EXPRESS SQLITE
Création d'une api REST en NODEjs avec les modules *express* et *SQLite*

## Installer le projet
Lancer les commandes suivantes pour compléter le projet
```bash
git clone http://...
npm install sqlite3
npm install express
```
